# Gem
 A simple, general-purpose and compiled programming language

## Features
- Compiled: Compiled to native C code, meaning it is fast and efficient
- Simple syntax: The syntax is simple and easy to learn, almost like a bridge between C and Python
- Can be extended with C: The language's standard library is written in C, so it is easy to extend the language using C code. You can also use the Gem language constructs from the Gem standard library in normal C code.
