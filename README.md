# Gem
A simple, speedy programming language

Gem is a high level programming language that is designed to be easy to learn and use, while still providing low level and high level features. It uses LLVM for compilation to generate efficient machine code. Gem aims to be easy and pleasant to use, with a focus on simplicity and readability. This means that memory management is handled automatically, allowing developers to focus on writing code without worrying about memory management but still able to access higher performance.
